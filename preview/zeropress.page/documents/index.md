# ZeroPress bundled themes

ZeroPress maintains a small set of bundled theme previews for `@zeropress/build-pages`.

Each preview is a standalone static site built with the same public Build Pages workflow. These are maintainer-published previews, not a public theme marketplace or third-party theme directory.

## Themes

### `zeropress.docs1`

Compact documentation theme for smaller documentation sites, package manuals, and reference pages that fit well in top navigation.

- Usage: `--theme docs1`
- Preview: [docs1.zeropress.page](https://docs1.zeropress.page/)
- Markdown examples: [docs1.zeropress.page/markdown-examples/](https://docs1.zeropress.page/markdown-examples/)
- Example source: [preview/docs1.zeropress.page](https://github.com/zeropress-app/zeropress.page/tree/main/preview/docs1.zeropress.page)
- Theme source: [themes/docs1](https://github.com/zeropress-app/zeropress-build-pages/tree/main/themes/docs1)

### `zeropress.docs2`

Sidebar documentation theme for larger documentation sites that need grouped navigation, command palette search, a page table of contents, and collection-based previous/next links.

- Usage: `--theme docs2`
- Preview: [build-pages.zeropress.dev](https://build-pages.zeropress.dev/)
- Markdown examples: [build-pages.zeropress.dev/guides/markdown/examples](https://build-pages.zeropress.dev/guides/markdown/examples)
- Example source: [build-pages.zeropress.dev repository](https://github.com/zeropress-app/build-pages.zeropress.dev)
- Theme source: [themes/docs2](https://github.com/zeropress-app/zeropress-build-pages/tree/main/themes/docs2)

### `zeropress.plain`

Minimal document theme for publishing standalone Markdown pages with as little site chrome as possible.

- Usage: `--theme plain`
- Preview: [plain.zeropress.page](https://plain.zeropress.page/)
- Markdown examples: [plain.zeropress.page/markdown-examples/](https://plain.zeropress.page/markdown-examples/)
- Example source: [preview/plain.zeropress.page](https://github.com/zeropress-app/zeropress.page/tree/main/preview/plain.zeropress.page)
- Theme source: [themes/plain](https://github.com/zeropress-app/zeropress-build-pages/tree/main/themes/plain)
