# Docs1 Theme Preview

This site previews `zeropress.docs1`, the compact documentation theme bundled with `@zeropress/build-pages`.

Docs1 is designed for small documentation sites, package manuals, release notes, and focused reference pages where a top navigation bar is enough to understand the site.

## What This Preview Shows

- top navigation with no sidebar
- static search UI
- generated page table of contents
- updated dates and source Markdown links
- collection-based previous and next links
- Markdown rendering for tables, alerts, code blocks, and Mermaid fences

## When Docs1 Fits

Use docs1 when the site has a small number of top-level pages and does not need a deep sidebar tree.

For larger manuals with many sections, use `zeropress.docs2` instead.

> [!NOTE]
> This preview asks crawlers not to index it and is maintained as a theme preview.
