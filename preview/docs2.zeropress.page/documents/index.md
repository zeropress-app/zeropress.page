# Docs2 Theme Preview

This site previews `zeropress.docs2`, the sidebar documentation theme bundled with `@zeropress/build-pages`.

Docs2 is designed for larger documentation sites, product manuals, and technical guides where grouped sidebar navigation makes the site easier to scan.

## What This Preview Shows

- grouped sidebar navigation
- one nested child level under a sidebar page
- static search UI
- generated page table of contents
- updated dates and source Markdown links
- collection-based previous and next links that stop at group boundaries
- Markdown rendering for tables, alerts, code blocks, and Mermaid fences

## When Docs2 Fits

Use docs2 when a documentation site needs grouped navigation, a command palette search experience, per-page table of contents, and a predictable previous/next reading flow.

For smaller sites where a top navigation bar is enough, use `zeropress.docs1` instead.

> [!NOTE]
> This preview asks crawlers not to index it and is maintained as a theme preview.
